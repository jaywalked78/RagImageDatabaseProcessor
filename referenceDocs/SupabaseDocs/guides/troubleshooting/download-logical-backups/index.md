# How to download logical backups in Supabase with physical backups enabled?

Last edited: 2/21/2025

* * *

If you're unable to download backups due to physical backups being enabled, you can use the Supabase CLI command `pgdump` as an alternative. For step-by-step instructions, check out our guide: [Backup & Restore](https://supabase.com/docs/guides/platform/migrating-within-supabase/backup-restore).

You can also learn more about how backups work in Supabase here: [Database Backups](https://supabase.com/docs/guides/platform/backups).
