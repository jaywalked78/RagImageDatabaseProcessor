# Why am I being redirected to the wrong url when using auth redirectTo option?

Last edited: 2/21/2025

* * *

Why are redirects going to localhost instead of the production site URL? In order for the provided `redirectTo` option to work you must set the exact URL in the shown Redirect URL's setting.

![image](https://supabase.com/docs/img/troubleshooting/224379580-fa77bd31-bb58-47e6-90ce-64e140f32579.png)

For more information on formats for redirect URL settings see the documentation here: [https://supabase.com/docs/guides/auth/overview#redirect-urls-and-wildcards](https://supabase.com/docs/guides/auth/overview#redirect-urls-and-wildcards)
